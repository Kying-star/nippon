# ない形

## 变形规则

```
一类动词：
う段　ー＞　あ段＋ない
    以う结尾的动词，要去掉う变成わない。買うー＞買わない
    ある的ない形就是ない

二类动词：去掉 る＋ない

三类动词：
    か变动词：来る　ー＞　来（こ）ない
    サ变动次：する　ー＞　しない

形容词：去掉 い＋くない

形容动词和名词：～＋では（じゃ）ない
```
