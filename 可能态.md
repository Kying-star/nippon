# 可能态

## 变形规则

```
一类动词：う段假名　ー＞　え段＋る

二类动词：去る＋られる

三类动词：
    か变动词：くる　ー＞　こられる
    サ变动词：する　ー＞　できる
```

## 变化后的词性

- 变化之后都是二类动词
- 变化之后都是自动词

## 含义

### 1.事物的性质

```
例：この野菜は生で食べらない。
译：这个蔬菜不能生吃。
```

### 2.表示可能性

```
例：この事は両親に言えません。
译：这件事不能对父母说。

例：今は社長に会えません。
译：现在无法会见社长。

例：事務所の人は誰もそのコンーピュータを使えます。
译：事务所的人谁都不会使用那台电脑。
```

### 3.能力

```
例：日本語を話せます。
译：会说日语。
```
